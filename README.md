# **App Name**: MunchMate

## Core Features:

- Dynamic Category Filtering: Enable filtering of menu items based on selected categories.
- Menu Recommendations: Dynamically tailor menu recommendations based on table location, previously selected filters, time of day, and overall eating habits to optimize user experience.
- Special Offers Display: Highlight special offers through a rotating banner within the app.
- Search Functionality: Allows users to quickly find the food they want using a search bar.
- Cart Management: Keep track of items as they're added to the cart for easy ordering.
- Owner Dashboard (Desktop): Provides a desktop dashboard for the owner to view orders, analytics, and manage payroll. Includes data visualization of sales trends and customer behavior.
- Kitchen Order View (Desktop): Presents a real-time order queue for the kitchen staff, indicating which items to cook and in what order. Includes features for marking orders as 'in progress' and 'completed'.

## Style Guidelines:

- Primary color: Fresh green (#22bb66) for vibrancy and connection to food and freshness.
- Background color: Soft white (#FAFAFA) to create a clean, open layout.
- Accent color: Muted orange (#E57373) to highlight discounts and special promotions.
- Font: 'Poppins' (sans-serif) for a modern, approachable, and readable interface.
- Use Lucide Icons for consistent and clean design throughout the app.
- Mobile-first design with rounded cards and generous spacing to provide a tap-friendly and visually appealing experience.
- Subtle scale-up animations on tapping food images, category icons, and other interactive elements to enhance user engagement.
